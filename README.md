# ChargeLog - Android 电池充放电实时监控工具

ChargeLog 是一款专为 Android 设备设计的轻量电池充放电数据记录与监控工具。
*(本项目由 AI 辅助开发)*

---

## 🌟 主要功能

* **实时指标显示**：在应用主界面实时展示设备当前的电压 (V)、电流 (A)、功率 (W) 及电量百分比 (%)。
* **后台监控与实时通知**：支持常驻后台服务，在系统通知栏秒级刷新显示当前的实时电池指标与记录状态（支持展开为多行展示），并适配了 Android 16 的 Live Update 状态栏胶囊与锁屏置顶显示。
* **充放电历史记录**：支持一键“开始”和“停止”记录充放电会话。历史记录按会话分组保存，可随时查阅历史曲线与原始数据表格，支持历史记录的单条删除与一键清空。

---

## 🚀 使用说明

1. **实时监测**：打开应用即可在看板和折叠态通知栏中实时看到当前设备的电压、电流、功率和电量指标。
2. **记录日志**：
   * 点击 **“开始”**：在后台开启采样，并开始向新的历史会话写入数据。
   * 点击 **“停止”**：停止采样，并将当前记录保存封存为历史记录。
   * 点击 **“清空屏幕”**：重置当前图表；如果正在记录中，确认后将丢弃本次采样的临时数据。
3. **管理历史**：点击 **“历史记录”** 查看列表，点击卡片可回放历史曲线并查阅原始数据，长按卡片可删除该条历史。
